# Rust tutotial

## Commands 

1. How to run all test cases in a module?
   
    Use command: `cargo test module_name`.
    Such as: 

```

    cargo test polygon

```

1. How to run all test cases in a workspace?
   
    Use command: `cargo test -- workspace_member_name xxxx`.
    Such as:

```

    cargo test xxxx

```
